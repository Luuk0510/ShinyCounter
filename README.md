# ShinyCounter
Personal Pokémon shiny counter app made in Flutter

## Features
- Lijst van Pokémon (custom + basis), alfabetisch, met caught-markering.
- Detailpagina met teller, +/-, caught-toggle en handmatige invoer.
- Android overlay (mini-counter) boven andere apps, met ± en sluitknop.
- Custom Pokémon toevoegen/bewerken/verwijderen incl. eigen afbeelding.
- Opslag in SharedPreferences; caught- en tellers persistent.
